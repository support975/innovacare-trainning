## 🎯 How to Use

### For New Users

1. **Visit Home Page**
   ```
   https://innovacare-training.web.app/home
   ```
   
2. **Click "Register to Your Organization"**
   
3. **Search and Select Organization**
   - Type organization name
   - Select from dropdown
   
4. **Fill Account Details**
   - Full Name
   - Email
   - Password (8+ characters)
   - Confirm Password
   
5. **Accept Terms & Submit**
   
6. **You're Registered!**
   - Automatic redirect to organization dashboard
   - Ready to explore courses and features

   - View welcome message

   - Access organization features

## 🛠️ Troubleshooting

### Issue: Page Not Loading
**Solution**:
1. Hard refresh browser (Ctrl+Shift+R on Windows, Cmd+Shift+R on Mac)
2. Clear browser cache
3. Try different browser
4. Check internet connection

### Issue: Form Not Submitting
**Solution**:
1. Check browser console (F12)
2. Verify all fields are filled correctly
3. Check internet connection
4. Try in incognito mode

### Issue: Can't Find Organization
**Solution**:
1. Try different search terms
2. Check organization name spelling
3. Make sure organization is marked as active in database
4. Contact administrator for organization setup

### Issue: Login Not Working
**Solution**:
1. Verify email and password are correct
2. Check internet connection
3. Try password reset
4. Clear browser cookies
5. Try incognito mode
